# InterviewPrepAI

## AI-Powered Interview Preparation Platform

InterviewPrepAI is a MERN stack application designed to help users prepare for interviews using AI-generated interview questions and session management.

---

# Features

## Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Profile Image Upload

---

## Session Management
- Create Session
- Get All Sessions
- Get Session By ID
- Delete Session

---

## Question Management
- Add Questions
- Add Answers
- Pin Questions
- Add Notes

---

# Tech Stack

## Backend

# Installation

## Clone Repository

```bash
git clone https://github.com/SunnyS315/Project-ASV.git
```

---

## Install Dependencies

```bash
npm install
```

---

## Create .env File

```env
PORT=8000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

---

## Run Server

```bash
npm run dev
```
